\begin{itemize}
  \item 
$(1)$ $1$不满足，$2$不满足，$3$满足， $4$不满足；
\item

\item $(3)$ 只要规则中不出现循环赋值，就存在。因此，$1$存在，$2$存在，$3$存在，而$4$中，$A.s = D.i = B.i + C.i = A.s+C.s+C.i$，所以不存在。
\end{itemize}
